# Customer_Behavior_Analysis

Overview

This project demonstrates an end-to-end data analytics workflow â€” from data loading and cleaning in Python, to exploratory data analysis (EDA), SQL querying, and visualization using PowerBI. Finally, insights are summarized in a business report and presentation. The objective is to extract actionable insights and communicate findings effectively through visuals.

Tools Used

Programming: 	Python(Pandas)
Database:	MySQL
Visualization:	Power BI
Presentation:	Gamma App

Project Steps

1. Load and Explore Data: Imported dataset using Pandas, Checked data shape, info, and basic statistics and Identified missing values and duplicates

2. Data Cleaning: Handled null values, removed duplicates, Standardized column names and data types and Filtered and transformed data for better analysis
   
3. SQL Integration: Imported cleaned dataset into MySQL performed queries for deeper insights, such as:
a) Top-selling categories
b) Regional sales performance
c) Customer segmentation and trends

4. Built an interactive Power Bi dashboard displaying:
a) Sales and profit overview
b) Regional and category performance
c) Top products and customer segments
d) Applied filters, slicers, and KPIs
